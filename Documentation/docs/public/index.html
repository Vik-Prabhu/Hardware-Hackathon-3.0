The Pipeline

How Green Node Works

🌱

Sense

ESP32-C3 nodes measure temperature, soil moisture, humidity, and light intensity in real time.

📶

Transmit

Data is pushed over WiFi to the FastAPI backend via HTTP POST with mDNS discovery.

🗄️

Store

The server logs readings in-memory with full historical timelines per node.

🧠

Analyze

Gemini AI processes sensor data + leaf images to produce structured health advisories.

📱

Act

The dashboard surfaces AI care plans, alerts, and multilingual voice guidance.
