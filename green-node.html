Sensor Stack
What Green Node Measures

A carefully selected sensor stack — no bloat, no missing pieces. Every component earns its place on the BOM.

🧠
ESP32-C3 — The Brain
Handles all logic, WiFi, BLE, and live OpenWeatherMap API calls. The intelligence hub of every node.
📷
ESP32-CAM — The Eyes
Daily snapshots for visual leaf health analysis. HSV colour thresholds detect yellowing, browning, and dark patches on-device.
🌡️
DHT11 × 2 — Dual Temp
Temperature and humidity at both canopy and soil level. Reveals actual thermal stress versus ambient air temperature.
☀️
LDR — Light Tracking
Light intensity, shading detection, and day/night cycle. Flags plants under flyovers or shade nets before they fail.
💧
Soil Moisture Sensor
The primary watering trigger. Combined with forecast data to eliminate pre-storm irrigation and catch post-sunny-week dryness.
📡
OTA Firmware Updates
esp-serial-flasher integration lets you push updates over WiFi to any deployed node. No technician visits required.

Analysis
SWOT Analysis
S
Strengths
  • Full node under ₹2,400 — deployable at scale without investor backing
  • Weather API integration is a genuine moat; no competitor combines on-ground data with forecast APIs
  • Route optimisation solves a worker problem — stronger sales argument than a plant problem
  • One hardware unit works across highways, nurseries, campuses, rooftops
  • Visual health monitoring at ₹350 — unmatched at this price point
W
Weaknesses
  • DHT11 ±2°C / ±5% RH accuracy degrades in sustained high-humidity; upgrade to DHT22 / SHT31 for production
  • Single soil sensor per node may miss variability across wider planting strips
  • No standalone power out of the box — solar + battery add ~₹1,100 per node
  • ESP-NOW range caps at ~200m; long highways need 4G relay modules
O
Opportunities
  • NHAI Smart Highway tenders — active government IoT procurement, no dedicated startup targeting horticulture
  • Nursery export compliance — EU/US phytosanitary documentation unlocks premium pricing
  • State PWD contracts — Karnataka, AP, Maharashtra have active urban greening budgets
  • CSR/ESG funded nodes for highway-adjacent companies
  • Research data licensing — microclimate corridor data has direct value for ISRO and climate consultancies
T
Threats
  • NHAI/PWD procurement takes 12–18 months; private nurseries are faster entry points
  • Highway vandalism — IP65 + tamper-evident mounting are non-negotiable
  • Connectivity dead zones — store-and-forward fallback essential for remote stretches
  • Indian monsoon stress at 85–95% RH for 4 months — conformal coating required from day one
